# LinkGlobal Network — Website

Standalone, static Home Page for LinkGlobal Network — a language-learning platform connecting learners with real tutors worldwide.

## Stack

- React (Vite) + TypeScript
- Tailwind CSS
- GSAP + ScrollTrigger
- Framer Motion
- Three.js via `react-globe.gl`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deploys automatically to Vercel on push to `main`.
